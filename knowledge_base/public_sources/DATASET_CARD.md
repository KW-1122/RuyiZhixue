# 公开数据说明

## 知识库数据

- 教育部《义务教育数学课程标准（2022年版）》：用于课程范围、核心素养和教学要求检索。
- 教育部《义务教育物理课程标准（2022年版）》：用于课程范围、核心素养和教学要求检索。
- 原始 PDF 保留在本目录。官方文件是扫描版，直接文本抽取为空，因此目前不进入检索库；后续需 OCR、人工抽检，再按页切片并保留页码来源。

## 独立评测数据

- C-Eval 初中数学、初中物理 dev/val/test 子集位于 `evaluation/public/ceval-exam`。
- 数据许可为 CC BY-NC-SA 4.0，仅用于非商业课程项目评测，不混入知识库，防止测试泄漏。

## 数据治理原则

- 权威课程标准进入知识库，公开题库仅用于独立评测。
- 所有生成回答必须展示来源；无法由证据支撑时拒答。
- 教师新增内容单独记录版本，不覆盖原始公开文档。
