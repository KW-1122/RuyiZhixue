# 如意智学知识来源与使用边界

更新时间：2026-07-22

## 1. 权威范围依据

### 义务教育数学课程标准（2022年版）

- 发布机构：中华人民共和国教育部
- 官方通知：https://www.moe.gov.cn/srcsite/A26/s8001/202204/t20220420_619921.html
- 官方 PDF：https://www.moe.gov.cn/srcsite/A26/s8001/202204/W020220420582346895190.pdf
- 本地存档：`义务教育数学课程标准2022.pdf`
- 用途：确定义务教育数学的内容范围、核心素养与学业要求。
- 使用边界：不大段复制原文；知识库内容采用团队原创解释，并在元数据中标注“依据课程标准整理”。

### 义务教育物理课程标准（2022年版）

- 发布机构：中华人民共和国教育部
- 官方通知：https://www.moe.gov.cn/srcsite/A26/s8001/202204/t20220420_619921.html
- 本地存档：`义务教育物理课程标准2022.pdf`
- 用途：确定初中物理主题、实验要求和跨学科实践范围。
- 使用边界：不复制商业教材；概念、公式、例题、易错点和练习由团队重新编写。

### 初中物理实验教学基本目录相关解读

- 发布机构：中华人民共和国教育部
- 官方页面：https://www.moe.gov.cn/jyb_xwfb/moe_2082/2023/2023_zl24/202311/t20231123_1091839.html
- 用途：确定实验类知识切片的覆盖方向。教育部解读说明目录包含68个基本实验，其中包括10个测量类、12个探究类必做实验。
- 使用边界：仅据此确定实验主题；实验步骤、安全提示和误差分析由团队原创。

## 2. 公开评测数据

### C-Eval

- 官方仓库：https://github.com/hkust-nlp/ceval
- 数据许可：CC BY-NC-SA 4.0
- 用途：只用于非商业课程项目的模型评测和测试集扩展。
- 使用边界：不将题库作为教学知识正文上传到Dify；保留署名、许可和引用信息。

## 3. 不直接入库的材料

- 商业出版社教材正文、教辅解析和付费题库。
- 国家中小学智慧教育平台的视频、课件和基础性作业原文件，除非对应页面明确提供可再分发许可。
- 无作者、无机构、无发布日期或无法追溯出处的网络文章。
- 模型生成但未经公式、单位、答案和来源核验的内容。

## 4. 入库内容标记

每个逻辑知识切片必须保存：`concept_id`、学科、年级、章节、切片类型、来源机构、依据文件、官方链接、内容责任方和复核状态。团队原创内容统一标注：

> 内容责任方：如意智学项目组原创整理；知识范围依据教育部《义务教育课程标准（2022年版）》；非课程标准原文。
