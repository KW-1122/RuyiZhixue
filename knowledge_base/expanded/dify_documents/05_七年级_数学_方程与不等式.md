## 等式性质与方程

concept_id: equation_basics
学科：数学
年级：七年级
章节：方程与不等式

### 概念
等式是用等号表示相等关系的式子，如\(a=b\)。方程是含有未知数的等式，如\(x+2=5\)。等式性质：性质1——等式两边同时加（或减）同一个数（或式子），所得结果仍是等式，即若\(a=b\)，则\(a\pm c = b\pm c\)；性质2——等式两边同时乘同一个数，或除以同一个不为0的数，所得结果仍是等式，即若\(a=b\)，则\(ac=bc\)，且当\(c\neq 0\)时，\(\frac{a}{c}=\frac{b}{c}\)。适用条件：解方程时，利用等式性质将方程变形，最终使未知数的系数化为1。

### 方法与公式
解方程的依据是等式性质。步骤：①在方程两边同时加或减同一个数（或式子），使含有未知数的项移到一边，常数项移到另一边；②在方程两边同时乘或除以同一个不为0的数，使未知数的系数变为1。注意：除以一个数等于乘这个数的倒数。移项时要注意改变符号，实际是等式性质1的应用。

### 原创例题
例题：解方程\(x+3=7\)。解析：利用等式性质1，两边同时减去3：\(x+3-3=7-3\)，得\(x=4\)。答案：\(x=4\)。

### 易错点
易错点1：移项不变号，如解\(x+3=7\)，错误移项为\(x=7+3\)，正确应为\(x=7-3\)。纠正：移项要变号。易错点2：等式两边同时除以一个数，忘记除数不能为0。易错点3：分数系数方程中，两边同乘分母时漏乘常数项。在解\(\frac{x}{2}=3\)时，两边同乘2得\(x=6\)，正确。

### 自检题
自检题：利用等式性质解方程\(3x=12\)。答案：\(x=4\)。解析：两边同时除以3得\(x=4\)。

## 一元一次方程及应用

concept_id: linear_equation
学科：数学
年级：七年级
章节：方程与不等式

### 概念
一元一次方程是指只含有一个未知数，未知数的次数都是1，且等号两边都是整式的方程。标准形式为\(ax+b=0\)（\(a\neq 0\)）。解一元一次方程的一般步骤：去分母（如果方程中有分母，两边同乘各分母的最小公倍数）、去括号（注意乘法分配律和符号变化）、移项（将含未知数的项移到左边，常数项移到右边，移项要变号）、合并同类项（将方程化为\(ax=b\)的形式）、系数化为1（两边同除以未知数的系数\(a\)）。应用：列方程解应用题时，先设未知数，找出等量关系，列出方程，解方程后检验并作答。

### 方法与公式
解一元一次方程的步骤：①去分母：方程两边同乘分母的最小公倍数，注意每一项都要乘，尤其不要漏乘不含分母的项。②去括号：用分配律去掉括号，注意括号前是负号时，括号内各项要变号。③移项：把含有未知数的项移到等号左边，常数项移到右边，移项要变号。④合并同类项：将方程化为\(ax=b\)形式。⑤系数化为1：两边同除以\(a\)得\(x=\frac{b}{a}\)。应用题的解法：审题设未知数，找等量关系列方程，解方程，检验答案是否符合实际。

### 原创例题
例题：解方程\(\frac{x+1}{2}=3\)。解析：去分母（两边乘2）：\(x+1=6\)；移项：\(x=6-1\)；合并：\(x=5\)。答案：\(x=5\)。

### 易错点
易错点1：去分母时漏乘不含分母的项，如方程\(\frac{x}{2}+1=3\)，错误只乘\(\frac{x}{2}\)而漏乘1和3。纠正：每一项都要乘2。易错点2：去括号时符号错误，如\(-3(x-2)\)误写为\(-3x-6\)，正确应为\(-3x+6\)。易错点3：移项时忘记变号，如\(x+3=5\)得\(x=5+3\)错误。纠正：移项要变号。

### 自检题
自检题：解方程\(2x-3=7\)。答案：\(x=5\)。解析：移项得\(2x=7+3=10\)，系数化为1得\(x=5\)。

## 不等式性质

concept_id: inequality_basics
学科：数学
年级：七年级
章节：方程与不等式

### 概念
不等式性质是在实数范围内研究不等式变形规则的基础。包括：性质1：如果a＞b，那么a±c＞b±c（不等式两边加或减同一个数，不等号方向不变）；性质2：如果a＞b，c＞0，那么ac＞bc，a/c＞b/c（乘以或除以正数，不等号方向不变）；性质3：如果a＞b，c＜0，那么ac＜bc，a/c＜b/c（乘以或除以负数，不等号方向改变）。这些性质适用于所有实数，是解一元一次不等式（组）的依据，也用于比较大小和证明不等式。注意：同乘（除）负数时不等号必须反向。

### 方法与公式
关键方法：利用不等式性质进行变形。公式：若a > b，则a + c > b + c；若a > b 且 c > 0，则ac > bc；若a > b 且 c < 0，则ac < bc。符号意义：>、<、≥、≤。步骤：1. 识别不等式；2. 确定要施加的运算（加、减、乘、除）；3. 检查运算数c的符号；4. 应用性质，若c<0则反转不等号；5. 化简结果。例如：解不等式-2x + 3 < 7。步骤：移项得-2x < 4，两边除以-2（负数），不等号反向得x > -2。

### 原创例题
【例题】若a > b，判断下列变形是否正确：① a-5 > b-5；② 3a > 3b；③ -a > -b；④ 2a+1 > 2b+1。【解析】①正确，性质1；②正确，性质2（c=3>0）；③错误，性质3（c=-1<0），应为-a < -b；④正确，先由性质2得2a>2b，再由性质1得2a+1>2b+1。【答案】①正确，②正确，③错误，④正确。
答案：计算结果或结论见上述分步解析。

### 易错点
易错点1：忘记不等号方向改变。当两边乘以或除以负数时，学生常忘记反向。纠正：强调“负号反向”，做练习时先写出性质再操作。易错点2：在叠加运算中混淆性质顺序。例如移项时误以为要反向。纠正：移项本质是加减运算，不等号不变；只有乘除负数才变向。易错点3：对分数或小数做乘除时不注意符号。纠正：判断数的符号优先，可先把小数化分数。

### 自检题
【自检题】解不等式：-3x + 6 ≤ 18，并将解集表示在数轴上。【答案】x ≥ -4。【解析】移项：-3x ≤ 12；除以-3（负数），不等号反向得x ≥ -4。数轴上在-4处画实心圆点，向右画线。

## 一元一次不等式与不等式组

concept_id: inequality
学科：数学
年级：七年级
章节：方程与不等式

### 概念
一元一次不等式是含有一个未知数，且未知数的最高次数为1的不等式，形如ax+b>0（a≠0）。一元一次不等式组是由几个相同未知数的一元一次不等式组成。解不等式组要求找出所有不等式解集的公共部分。适用条件：未知数系数不为零，且不等式经过化简后次数为1。解集常用数轴表示，端点注意实心（≥、≤）和空心（>、<）。不等式组取解集口诀：同大取大，同小取小，大小小大中间找，大大小小找不到。

### 方法与公式
关键方法：分别解每个不等式，再取公共部分。步骤：1. 去分母（注意乘正负数）；2. 去括号；3. 移项；4. 合并同类项；5. 系数化为1（注意变号）；6. 在数轴上表示各解集；7. 找出重叠部分即不等式组的解集。公式：解一元一次不等式与解方程类似，仅乘除负数时不等号反向。例如：解不等式组 {2x-1>3，x+2≤5}，由第一个得x>2，第二个得x≤3，解集为2<x≤3。

### 原创例题
【例题】解不等式组： { 3(x-2) ≥ x-4 ， (2x+1)/3 > x-1 }。【解析】1. 解第一个不等式：3x-6 ≥ x-4 → 2x ≥ 2 → x ≥ 1。2. 解第二个：2x+1 > 3x-3 → -x > -4 → x < 4（注意除以-1反向）。3. 在数轴上表示：x≥1用实心向右，x<4用空心向左，公共部分为1≤x<4。【答案】解集为 1 ≤ x < 4。
答案：计算结果或结论见上述分步解析。

### 易错点
易错点1：去分母时漏乘常数项。如(2x+1)/3 > x-1，漏乘3得2x+1>3x-1，忘记右边每项都乘3。纠正：去分母时每一项都乘以分母的最小公倍数。易错点2：系数化为1时忘记变号。例如-x > -4，错误得x > 4。纠正：根据性质3，当除以负数时，不等号反向。易错点3：解集求并集而不是交集。不等式组要求公共部分，学生有时错误地合并所有范围。纠正：强调用数轴找重叠部分。

### 自检题
【自检题】解不等式组：{4x-1>2x+5，3x+2>5x-4}。【答案】无解。【解析】第一个不等式解得x>3；第二个不等式移项得-2x>-6，两边除以-2得x<3。所以解集为x>3与x<3，无公共部分，故不等式组无解。
