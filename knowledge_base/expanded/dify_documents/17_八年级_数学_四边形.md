## 平行四边形判定与性质

concept_id: parallelogram
学科：数学
年级：八年级
章节：四边形

### 概念
平行四边形是两组对边分别平行的四边形。其性质包括：对边平行且相等；对角相等；邻角互补；对角线互相平分。判定方法有五种：①两组对边分别平行；②两组对边分别相等；③一组对边平行且相等；④两组对角分别相等；⑤对角线互相平分。这些性质与判定在解决几何问题时需灵活运用，注意区分条件与结论。

### 方法与公式
关键方法：1. 利用平行四边形性质求解边长或角度，如对边相等、对角相等。2. 通过判定条件证明四边形为平行四边形。常用公式：平行四边形面积=底×高，用字母表示为\(S = ah\)，其中\(a\)为底边长，\(h\)为高。步骤：先识别已知条件，选择对应性质或判定，必要时添加辅助线构造平行四边形。符号：\(\square ABCD\)表示平行四边形。

### 原创例题
例题：在平行四边形\(ABCD\)中，对角线\(AC\)与\(BD\)交于点\(O\)，\(E\)、\(F\)分别是\(OA\)、\(OC\)的中点。求证：四边形\(EBFD\)是平行四边形。
解析：由平行四边形性质，\(OA=OC\)，\(OB=OD\)。因为\(E\)、\(F\)是中点，所以\(OE = \frac{1}{2}OA = \frac{1}{2}OC = OF\)，即\(OE=OF\)。又\(OB=OD\)，所以对角线互相平分，四边形\(EBFD\)是平行四边形。
答案：证明过程如上。

### 易错点
易错点1：混淆性质与判定，如用对角相等证明对边平行（需先用其他条件）。纠正：明确性质是已知平行四边形推结论，判定是已知条件证平行四边形。易错点2：忽略中点条件，线段比例错误。纠正：仔细标注中点，利用线段倍分关系。易错点3：对角线互相平分条件不全，只证一半。纠正：需证两条对角线各自被平分。

### 自检题
自检题：在\(\square ABCD\)中，\(\angle A = 60^\circ\)，\(AB=4\)，\(AD=6\)，求\(\angle B\)和\(BC\)的长。
答案：\(\angle B = 120^\circ\)，\(BC=6\)。
解析：平行四边形邻角互补，所以\(\angle B = 180^\circ - 60^\circ = 120^\circ\)；对边相等，\(BC=AD=6\)。

## 矩形、菱形与正方形

concept_id: special_quadrilateral
学科：数学
年级：八年级
章节：四边形

### 概念
矩形、菱形和正方形是特殊的平行四边形。矩形定义：有一个角是直角的平行四边形；性质：对角线相等，四个角都是90°。菱形定义：有一组邻边相等的平行四边形；性质：对角线互相垂直平分且平分一组对角。正方形定义：邻边相等且有一个角是直角的平行四边形；性质：兼具矩形和菱形的所有性质。判定时需从定义出发，结合边、角、对角线条件。

### 方法与公式
关键方法：1. 矩形判定：先证平行四边形，再加一个直角或对角线相等。2. 菱形判定：先证平行四边形，再加一组邻边相等或对角线垂直。3. 正方形判定：先证矩形，再加一组邻边相等；或先证菱形，再加一个直角。常用公式：矩形面积=长×宽，菱形面积=对角线乘积的一半=\(\frac{1}{2}d_1d_2\)，正方形面积=边长\(^2\)。步骤：分析图形条件，选择正确判定路径，避免循环论证。

### 原创例题
例题：如图，在矩形\(ABCD\)中，\(E\)、\(F\)分别是\(AD\)、\(BC\)上的点，且\(AE=CF\)，连接\(EF\)交对角线\(BD\)于点\(O\)。求证：\(OE=OF\)。
解析：矩形\(ABCD\)中，\(AD\parallel BC\)，所以\(\angle EDO = \angle FBO\)，\(\angle DEO = \angle BFO\)。又\(AE=CF\)，且\(AD=BC\)，所以\(ED=BF\)。因此\(\triangle EDO \cong \triangle FBO\)（ASA），故\(OE=OF\)。
答案：证明过程如上。

### 易错点
易错点1：误认为矩形对角线互相垂直，菱形对角线相等。纠正：记清性质：矩形对角线相等，菱形对角线垂直。易错点2：正方形判定中，只满足矩形和菱形之一就下结论。纠正：必须同时满足两个条件。易错点3：在证明菱形时，只证四条边相等而不先证平行四边形。纠正：先证四边形是平行四边形，再证邻边相等更严谨。

### 自检题
自检题：菱形\(ABCD\)的周长为24，对角线\(AC=6\)，求另一条对角线\(BD\)的长和菱形面积。
答案：\(BD = 6\sqrt{3}\)，面积\(=18\sqrt{3}\)。
解析：菱形边长\(=24\div4=6\)，对角线互相垂直平分，设\(AC\)一半为3，边长6，由勾股定理得\(BO = \sqrt{6^2-3^2}=3\sqrt{3}\)，所以\(BD=6\sqrt{3}\)，面积\(=\frac{1}{2}\times6\times6\sqrt{3}=18\sqrt{3}\)。
