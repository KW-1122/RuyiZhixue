## 相似三角形判定与性质

concept_id: similar_triangle
学科：数学
年级：九年级
章节：相似与锐角三角函数

### 概念
相似三角形是形状相同但大小可能不同的三角形，对应角相等且对应边成比例。判定两个三角形相似的主要方法有：两角分别相等（AA）、两边对应成比例且夹角相等（SAS）、三边对应成比例（SSS）。直角三角形中，斜边和一条直角边对应成比例（HL）也可判定相似。相似三角形的性质：对应角相等，对应边成比例，周长比等于相似比，面积比等于相似比的平方。适用条件：已知两组角相等或两边成比例且夹角相等或三边成比例。

### 方法与公式
判定相似时，优先寻找角相等的条件，如平行线、公共角、等腰三角形等。用AA判定最简洁。已知两边成比例且夹角相等用SAS；已知三边成比例用SSS。常用符号“∼”表示相似。关键公式：相似比 \(k = \frac{\text{对应边1}}{\text{对应边2}}\)，面积比 \(= k^2\)。解题步骤：①确定相似三角形；②列比例式，注意对应顶点顺序；③代入已知值求解未知边或面积。添加辅助线构造相似三角形是常见技巧。

### 原创例题
在\(\triangle ABC\)中，点\(D\)在边\(AB\)上，\(\angle ACD = \angle B\)。若\(AD = 4\)，\(AC = 6\)，求\(AB\)的长。
解：∵\(\angle ACD = \angle B\)，且\(\angle A\)是公共角，∴\(\triangle ACD \sim \triangle ABC\)（AA）。由相似得\(\frac{AC}{AB} = \frac{AD}{AC}\)，即\(\frac{6}{AB} = \frac{4}{6}\)，解得\(AB = 9\)。
答案：\(AB = 9\)。

### 易错点
易错点1：列比例时对应边混乱。例如误写\(\frac{AD}{AC} = \frac{AB}{AC}\)。纠正方法：先明确相似三角形的对应顶点，按顺序写比例。
易错点2：忽略面积比等于相似比的平方。例如计算面积时直接使用相似比。纠正方法：牢记面积比是相似比的平方，注意平方运算。
易错点3：只考虑边成比例而忽略夹角相等，错误使用SAS判定。纠正方法：SAS必须指明夹角相等，且对应边成比例。

### 自检题
在\(\triangle ABC\)中，\(DE \parallel BC\)，\(AD:DB = 2:3\)，\(\triangle ADE\)的面积为8。求四边形\(DBCE\)的面积。
答案：42。
解析：由\(DE\parallel BC\)得\(\triangle ADE \sim \triangle ABC\)，相似比\(k = AD:AB = 2:5\)，面积比\(4:25\)。设\(\triangle ABC\)面积为\(S\)，则\(8:S = 4:25\)，得\(S=50\)。四边形面积\(=50-8=42\)。

## 锐角三角函数

concept_id: trigonometry
学科：数学
年级：九年级
章节：相似与锐角三角函数

### 概念
锐角三角函数是在直角三角形中定义的：对于锐角\(A\)，\(\sin A = \frac{\text{对边}}{\text{斜边}}\)，\(\cos A = \frac{\text{邻边}}{\text{斜边}}\)，\(\tan A = \frac{\text{对边}}{\text{邻边}}\)。其中对边和邻边相对于角\(A\)而言。锐角三角函数值只与角度大小有关，与三角形边长无关。取值范围：\(0 < \sin A < 1\)，\(0 < \cos A < 1\)，\(\tan A > 0\)。需熟记30°、45°、60°角的三角函数值。适用条件：在直角三角形中计算或已知一边一角求其他边角。

### 方法与公式
求锐角三角函数值的关键是将锐角放在直角三角形中，明确对边、邻边和斜边。常用关系：\(\sin^2 A + \cos^2 A = 1\)，\(\tan A = \frac{\sin A}{\cos A}\)。解直角三角形时，根据已知边角选择合适函数：已知斜边和角求对边用正弦，求邻边用余弦；已知角和邻边求对边用正切。步骤：①画图并标出已知边角；②选择三角函数关系式；③建立方程求解，注意单位统一。特殊角值可通过构造直角三角形记忆。

### 原创例题
在Rt\(\triangle ABC\)中，\(\angle C = 90^\circ\)，\(BC = 5\)，\(AB = 13\)，求\(\sin A\)，\(\cos A\)，\(\tan A\)。
解：由勾股定理得\(AC = \sqrt{13^2 - 5^2} = 12\)。
\(\sin A = \frac{BC}{AB} = \frac{5}{13}\)，\(\cos A = \frac{AC}{AB} = \frac{12}{13}\)，\(\tan A = \frac{BC}{AC} = \frac{5}{12}\)。
答案：\(\sin A = \frac{5}{13}\)，\(\cos A = \frac{12}{13}\)，\(\tan A = \frac{5}{12}\)。

### 易错点
易错点1：混淆对边与邻边。例如求\(\sin A\)时误用邻边。纠正方法：以角为基准，对边是角所对的边，邻边是角相邻的直角边。
易错点2：忘记用勾股定理求第三边就直接写比值。纠正方法：先求出所有边长再代入三角函数定义。
易错点3：特殊角三角函数值记忆错误，如\(\sin 30^\circ=\frac{1}{2}\)误为\(\frac{\sqrt{3}}{2}\)。纠正方法：通过画含特殊角的直角三角形辅助记忆。

### 自检题
在Rt\(\triangle ABC\)中，\(\angle C = 90^\circ\)，\(\sin A = \frac{3}{5}\)，\(BC = 6\)，求\(AB\)和\(AC\)。
答案：\(AB = 10\)，\(AC = 8\)。
解析：由\(\sin A = \frac{BC}{AB}\)得\(\frac{6}{AB} = \frac{3}{5}\)，解得\(AB = 10\)。再由勾股定理\(AC = \sqrt{10^2 - 6^2} = 8\)。
