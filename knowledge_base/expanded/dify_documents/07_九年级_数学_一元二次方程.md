## 一元二次方程解法

concept_id: quadratic_equation
学科：数学
年级：九年级
章节：一元二次方程

### 概念
一元二次方程是形如\( ax^2+bx+c=0 \)（\( a\neq0 \)）的整式方程。解法有四种：直接开平方法（适用于\( x^2=p \)形式）、配方法（化为完全平方式）、公式法（利用求根公式\( x=\frac{-b\pm\sqrt{b^2-4ac}}{2a} \)）和因式分解法（左边分解为两个一次因式乘积）。每种方法适用条件不同，公式法是通用解法，需注意判别式\(\Delta=b^2-4ac\)决定根的情况。

### 方法与公式
1. 直接开平方法：\( x^2 = p \)，则\( x = \pm\sqrt{p} \)（\( p\geq0 \)）。
2. 配方法：移常数项，两边加\((\frac{b}{2a})^2\)，写成\((x+\frac{b}{2a})^2 = \frac{b^2-4ac}{4a^2}\)，再开平方。
3. 公式法：先计算\(\Delta = b^2-4ac\)；若\(\Delta\geq0\)，则\( x = \frac{-b\pm\sqrt{\Delta}}{2a} \)；若\(\Delta<0\)，无实数根。
4. 因式分解法：将方程化为\( (mx+n)(px+q)=0 \)，则\( mx+n=0 \)或\( px+q=0 \)。步骤：去括号、移项、合并同类项，选择合适方法。

### 原创例题
例题：解方程\( 2x^2 - 5x + 2 = 0 \)（用两种方法）。
解析：
方法一（公式法）：\( a=2, b=-5, c=2 \)，\(\Delta = (-5)^2 - 4\times2\times2 = 25-16=9>0 \)，\( x = \frac{5\pm\sqrt{9}}{2\times2} = \frac{5\pm3}{4} \)，\( x_1=2, x_2=0.5 \)。
方法二（因式分解法）：十字相乘法：\( 2x^2 -5x +2 = (2x-1)(x-2)=0 \)，得\( 2x-1=0 \)或\( x-2=0 \)，\( x_1=0.5, x_2=2 \)。
答案：\( x_1=2, x_2=\frac{1}{2} \)。

### 易错点
1. 求根公式中符号错误：漏掉负号或分子中\(-b\)写成\(b\)。纠正：牢记公式\( x=\frac{-b\pm\sqrt{b^2-4ac}}{2a} \)。
2. 忽略判别式符号直接开方：如认为\(\sqrt{\Delta}\)总是有意义。纠正：先计算\(\Delta\)，若小于0则无实数根。
3. 因式分解时系数处理错误：如\(2x^2-5x+2\)分解为\((2x-1)(x-2)\)，注意符号。纠正：通过展开验证。

### 自检题
自检题：解方程\( x^2 - 4x - 5 = 0 \)（用配方法或公式法）。
答案：\( x_1=5, x_2=-1 \)。
解析：配方法：移项\( x^2-4x=5 \)，加4得\( (x-2)^2=9 \)，开方得\( x-2=\pm3 \)，\( x=5 \)或\( x=-1 \)。公式法：\(\Delta=16+20=36\)，\( x=\frac{4\pm6}{2} \)，解同上。

## 一元二次方程应用

concept_id: quadratic_equation_application
学科：数学
年级：九年级
章节：一元二次方程

### 概念
一元二次方程应用是指利用一元二次方程解决实际问题，常见模型包括面积、增长率、利润、行程等。核心是将实际问题中的等量关系抽象为二次方程\(ax^2+bx+c=0\)的形式。适用条件：问题中存在未知数，且等量关系能表示为含未知数的二次式。注意：解方程后需根据实际意义取舍，如长度、人数等非负量。

### 方法与公式
关键方法：审题设未知数，寻找等量关系列方程，化为标准形\(ax^2+bx+c=0\)。常用解法：公式法\(x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}\)（判别式\(\Delta=b^2-4ac\)）或因式分解法。步骤：1. 设未知数，明确单位；2. 由题意列等式；3. 整理为一般式；4. 求解方程；5. 检验解的合理性并作答。

### 原创例题
【例题】某矩形花圃长比宽多3米，面积为40平方米，求长和宽。\n解：设宽为\(x\)米，则长为\((x+3)\)米。由面积公式得\(x(x+3)=40\)，即\(x^2+3x-40=0\)。因式分解\((x+8)(x-5)=0\)，解得\(x=5\)或\(x=-8\)（舍去负根）。故宽5米，长8米。答：花圃长8米，宽5米。

### 易错点
易错点：1. 忽略实际意义，如未舍去负根。纠正：检查解是否符合非负、整数等条件。2. 单位不统一，如长度用米，面积用平方厘米。纠正：统一单位后再列方程。3. 设未知数时未明确表示什么。纠正：用文字描述“设……为\(x\)”。

### 自检题
【自检题】某商品原价100元，连续两次降价后售价64元，求平均每次降价率。\n答案：平均降价率20%。\n解析：设降价率\(x\)，则\(100(1-x)^2=64\)，得\((1-x)^2=0.64\)，\(1-x=\pm0.8\)，解得\(x=0.2\)或\(x=1.8\)（舍），故降价率20%。
