# 如意智学运行部署手册

## 本机一键运行

PowerShell 执行：

```powershell
.\run.ps1
```

脚本会创建虚拟环境、安装依赖、执行环境诊断并启动服务。访问 `http://127.0.0.1:8000`。

## Docker 运行

```powershell
docker compose up --build -d
docker compose ps
```

`data` 目录作为 SQLite 持久卷。健康检查接口为 `/api/health`。

## Dify 模式

1. 复制 `.env.example` 为 `.env`。
2. 启动 Dify 并填写应用与知识库 API Key。
3. 运行 `python scripts/upload_to_dify.py`。
4. 把脚本输出的 Dataset ID 写入 `.env`。
5. 按 `dify/workflow_blueprint.json` 在画布配置工作流。
6. 重启如意智学，左下角应显示 Dify RAG 引擎。

## 答辩故障降级

- Dify 不可用：自动切换本地 79 切片 GraphRAG。
- OCR 不可用：图片上传后允许人工校正题干。
- 网络不可用：核心字体会使用系统字体，所有业务接口仍在本地。
- 端口占用：修改 `.env` 中 `PORT`。

## 答辩前检查

```powershell
python scripts/diagnose.py
python -m unittest discover -s tests -v
python evaluation/run_evaluation.py
```

可选安装真实本地语义模型环境：`pip install -r requirements-ml-optional.txt`。模型首次使用需要下载；无网络时继续使用字符语义混合检索和 Dify 向量库。OCR 基准样本放入 `evaluation/samples/` 后运行 `python evaluation/run_ocr_benchmark.py`。

数据库备份：`python scripts/backup_database.py`。SQLite 使用 WAL 和 10 秒忙等待，支持课程演示级并发。
