# 如意智学高分答辩 PPT 逐页稿（12 页 / 10–15 分钟）

## 1. 封面（20 秒）

如意智学：面向中小学教学的 Dify + GraphRAG 智能辅导助手。副标题突出“有依据地教、分步骤地引导、可量化地评估”。

## 2. 教育痛点（50 秒）

普通搜索只给答案；通用大模型可能幻觉；教师难以实时观察每位学生。用一个错误答案截图与一个带引用回答形成对比。

## 3. 产品定位（50 秒）

学生获得启发辅导、图片题目、学习路径；教师获得班级薄弱点、低置信度复核和反馈闭环。

## 4. 总体架构（70 秒）

资料与元数据 → Dify 混合检索 → 两跳知识图谱扩展 → 教学 Prompt → 证据/算式/单位核验 → SQLite 学习档案。

## 5. 知识工程（60 秒）

展示 79 个切片、数学物理双学科、四类切片结构和来源字段；解释为什么按“概念、例题、易错、练习”拆分。

## 6. Dify 工作流（70 秒）

问题分类、混合检索、重排、教学生成、答案核验和引用输出。展示 Top K=5、阈值 0.25 等真实参数。

## 7. GraphRAG 创新（70 秒）

展示修正后的双泳道图谱。以“浮力”为例：力 → 压强 → 浮力；说明两跳扩展和个性化学习路径。

## 8. 防幻觉与安全（60 秒）

介绍证据覆盖、低置信度、算式核验、单位检查、超范围拒答和危险内容守护。

## 9. 产品演示（2–3 分钟）

上传题目或选择示例 → 先提示 → 查看完整讲解 → 查看引用和学习路径 → 收藏 → 教师端查看记录。

## 10. 实验结果（70 秒）

79 个知识切片，23 个检索评测问题，Recall@3=91.3%，MRR=91.3%，超范围拒答与安全守护通过，10 项单元测试通过。只展示真实运行结果。

## 11. 工程与团队（50 秒）

Docker、健康检查、SQLite、Dify/本地双引擎；列出产品、数据、RAG、前后端、测试答辩的成员贡献和 Git 证据。

## 12. 总结与路线图（30 秒）

已经形成“问、学、练、评、管”闭环。后续扩展更多学科、教师自定义题库和校级部署。

## 备用问答

- 为什么不用纯大模型？答：知识时效、可追溯和幻觉风险。
- 图谱有什么真实作用？答：两跳扩展、前置诊断与学习路径，不只是展示。
- 91.3% 如何得到？答：固定评测集、Recall@3 和 MRR 脚本可复现。
- Dify 挂了怎么办？答：本地 GraphRAG 自动降级，演示链路不中断。
